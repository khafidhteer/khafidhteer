## 🚀 Featured Projects

Here are some of my open-source repositories that demonstrate my capabilities:

<h3>🧠 <a href="https://github.com/khafidhteer/blackbox-automl" target="_blank">BlackBox AutoML</a></h3>

An automated machine learning pipeline powered by AutoGluon. Drop a CSV file and receive a fully documented Jupyter Notebook covering data cleaning, model training (LinearModel, XGBoost, LightGBM, Random Forest, ensembles), evaluation, and recommendations. Supports lightweight, balanced, and best-quality presets with Docker deployment.

> *Tags: autogluon, automl, python, jupyter-notebook*

<h3>🚗 <a href="https://github.com/khafidhteer/car-resale-analysis" target="_blank">Car Resale Value Analysis</a></h3>

A comprehensive market analysis of Toyota's used car inventory. Includes rigorous EDA, outlier removal, Random Forest modeling (R² 0.96), market segmentation strategy (Premium/Eco/Budget), and identification of "Underpriced Gems" — vehicles with immediate high-margin opportunities.

> *Tags: data-science, exploratory-data-analysis, python, jupyter-notebook*

<h3>🔗 <a href="https://github.com/khafidhteer/company-similarity" target="_blank">Company Name Matching Tool</a></h3>

A Python CLI tool that matches company names from input CSVs against master lists using `rapidfuzz` fuzzy matching. Features streamed CSV processing for low memory usage, real-time progress tracking with ETA, Docker-ready execution, and up to 5 top matches with confidence scores.

> *Tags: python, similarity, cli, docker*

<h3>📄 <a href="https://github.com/khafidhteer/webdocs2md" target="_blank">webdocs2md</a></h3>

Crawl documentation websites and compile them into a single structured Markdown document. Features BFS crawling, smart content extraction, Playwright for JS-rendered sites, and a GitHub Profile Summary mode that fetches all public repository READMEs with aggregated stats.

> *Tags: python, web-scraping, documentation, markdown, crawling*

<h3>🔄 <a href="https://github.com/khafidhteer/json2csv" target="_blank">json2csv</a></h3>

Convert JSON files to structured CSV tables with automatic nested object flattening and array expansion. Supports batch processing, multiple JSON formats, Docker deployment, and has zero external dependencies beyond Python standard library.

> *Tags: python, json, csv, data-transformation*
